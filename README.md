# L5XJS

A simple API designed to build and parse Allen Bradley *.L5X* files in order to empower users to automate PLC code generation.

## Prerequisites

latest version of [NodeJS](https://nodejs.org/en/)

## Getting Started

Install with npm

```
npm install l5x-js --save
```
